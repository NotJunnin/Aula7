Como subir o projeto para o github

Projeto já iniciado

Passo 1: git init

Passo 2: git checkout -b main

Passo 3: git remote add origin "url"

Passo 4: add .

Passo 5: commit -m "mensagem"

Passo 6: push origin main

Passo 2: git add .

Passo 3: git commit -m "mensagem"
